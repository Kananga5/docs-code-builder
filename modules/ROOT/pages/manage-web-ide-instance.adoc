Anypoint Code Builder enables you to manage your cloud IDE instances directly from the central page.

image::web-ide-management-central-page.png[]

[calloutlist]
. *Version and IDE Details*: +
Shows the version of your Anypoint Code Builder cloud IDE instance. You can additionally hover over *View IDE Details* for a few internal details of your cloud IDE instance:
* *Image*: The ID of the image of your cloud IDE instance.
* *URL*: The URL to directly access your cloud IDE instance. +
This URL is necessary to ping your locally deployed Mule application when debugging.
* *Created at*: Date and time of the creation of the cloud IDE instance.
. *Reboot*: Reboots your cloud IDE instance.
. *Delete*: Deletes your cloud IDE instance.
+
[WARNING]
--
Deleting your cloud IDE cannot be undone and you will lose all the projects hosted on that instance.
--
