= Send Feedback

include::reuse::partial$beta-banner.adoc[tag="anypoint-code-builder"]

You can send us feedback or report issues concerning your experience using Anypoint Code Builder.

For questions or general discussions about Code Builder, see https://trailhead.salesforce.com/trailblazer-community/topics/mulesoftanypointcodebuilder?sort=LAST_MODIFIED_DATE_DESC[Trailblazer Discussion Topic^].

To provide feedback or report an issue:

. Launch Anypoint Code Builder web IDE.
. In your Anypoint Code Builder welcome mat, select *Submit your feedback through GitHub*.
+
image::submit-feedback.png[Feedback link highlighted in the welcome mat view.]
. Select *New Issue*.
+
image::provide-feedback-new-issue.png[New Issue button highlighted in the issues creation page.]
. Provide a title for your feedback or issue, and use the comment box to explain your feedback.
. Select *Submit new issue*.
+
image::provide-feedback-submit-issue.png["Title, Comment Box, and Submit button highlighted in the issues creation page."]


[NOTE]
You must follow our https://github.com/forcedotcom/try-acb-feedback/blob/main/CODE_OF_CONDUCT.adoc[Community Code of Conduct^] when submitting feedback.
