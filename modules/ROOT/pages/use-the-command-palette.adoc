The Command Palette in Anypoint Code Builder is the control center for all commands you use to perform tasks. Knowing how to open it is important for your productivity.

. Open the Command Palette in one of the following ways:

* From the desktop or cloud IDE, press Cmd+Shift+p (Mac) or Ctrl+Shift+p (Windows).
* From the desktop IDE, select *View* > *Command Palette*. 
* From the cloud IDE, click the menu icon, and select *View* > *Command Palette*:
+
image::open-command-palette.png[]

. Type `MuleSoft:` and browse all the available commands:
+
image::all-mulesoft-commands.png[]
